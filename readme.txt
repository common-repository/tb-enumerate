Learn the numbering system by position. 
Shortcode: [tbenumerate]

== Description ==

This plugin allows you to discover the numbering system by position invented by the Hindus, that every day is used unconsciously most of the time. The base recognized by this plugin are from 2 to 26. You can integrate this plugin in your wordpress page with the shortcode [tbenumerate]

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
